# Kubikovač guľatiny – inštalačné balíky

Program na výpočet objemu guľatiny meranej s kôrou, kompatibilný s STN 48 0009.
Tu sú len hotové **inštalačné balíky** (zdrojový kód je súkromný).

## Stiahnutie najnovšej verzie

👉 **[Releases](../../releases/latest)** – vyberte podľa systému:

| Systém | Súbor | Inštalácia |
|---|---|---|
| Windows | `KubikovacSetup-*.exe` | dvojklik → Ďalej → Dokončiť |
| Android | `Kubikovac-*.apk` | nainštalovať cez existujúcu appku |
| Linux (Ubuntu/Debian) | `kubikovac_*_amd64.deb` | `sudo apt install ./kubikovac_*.deb` |

Program vás na novú verziu upozorní aj sám pri spustení.
